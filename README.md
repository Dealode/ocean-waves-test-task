# Тестове завдання:
## 1. Зібрати сцену в Unity HDRP
- Версія Unity: 2021.3 LTS або вище
- Налаштувати HDRP-проект із Crest Ocean System
- Додати реалістичну водну поверхню з хвилями та прозорістю

## 2. Інтеграція DWP
- Додати плаваючий об'єкт (човен, платформа, скринька)
- Налаштувати фізику: маса, центр тяжкості, плавучість
- Об'єкт повинен реагувати на хвилі Crest та корректно плавати

## 3. Контроль руху
- Додати керування човном: клавіші WASD або Arrow Keys
- Реалізувати реалістичний опір води та інерцію

## 4. Візуалізація та продуктивність
- Налаштувати камеру з постобробкою (Depth of Field, Bloom)
- Додати LOD або оптимізацію, якщо сцена перевантажена
- Складання .exe має запускатися без помилок

## 5. (Опціонально) Інтеграція з ML-Agents
- Додати CameraSensorComponent на човен
- Підготувати BehaviorParameters для навчання руху
- Сцена має запускатися через UnityEnvironment

## Що необхідно надати:
.unitypackage або .zip проекту
Складання .exe з човном на воді
Короткий опис архітектури: які компоненти використані, як пов'язані Crest та DWP
(Якщо ML-Agents) - debug_env.py та config.yaml
